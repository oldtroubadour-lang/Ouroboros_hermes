# Ouroboros_hermes
Этот проект представляет собой Систему поддержки принятия решений (СППР) банка на базе автономного агента Ouroboros. Система в реальном времени мониторит чрезвычайные ситуации, сопоставляет их с геосеткой банковских отделений/банкоматов, формирует тактические сценарии реагирования и высылает интерактивный отчет руководству в Telegram.

(1) Структура файлов и их размещение
Для успешного запуска перенесите файлы из репозитория на новую машину в следующие папки:

  А. Модули логики СППР (кастомные навыки)
  Все папки навыков из репозитория GitHub скопируйте в каталог пользовательских расширений вашего агента Ouroboros:
👉 Путь: ~/Ouroboros/data/skills/external/
- 
- temp_storage — общая память и Episodic DB системы.
- incident_ner — сборщик новостей МЧС и классификатор инцидентов.
- asset_geo_matcher — пространственный сопоставитель локаций ЧС и объектов банка.
- risk_scenario_generator — планировщик тактических сценариев.
- telegram-bridge — интерактивный бот управления Hermes.
- dss_orchestrator - оркестратор и организатор совместной работы остальных навыков

Б. Справочники (Активы и Сценарии)
Файлы Excel, находящиеся в корне репозитория GitHub, скопируйте в стандартную папку загрузок текущего системного пользователя:
👉 Путь: C:\Users\<Имя_Пользователя>\Downloads\ (на Windows) или ~/Downloads/ (на macOS/Linux).
- Справочник объектов для агента.xlsx — географический реестр 20 758 банкоматов и офисов (фиктивные данные, все совпадения случайные)
- Агент - примеры сценариев.xlsx — примеры реагирования и решений (в будущем будет RAG)


(2) Быстрый старт (5 простых шагов)
Шаг 1. Установка платформы
Установите и запустите Ouroboros на новой машине, если еще не установлен и не настроен. 

Шаг 2. Развертывание зависимостей
Для обработки Excel-справочников в окружение Ouroboros должны быть установлены внешние библиотеки. Откройте терминал на новой машине и выполните:
pip install openpyxl edge-tts

Шаг 3. Ввод Telegram-секретов
Создайте бота в telegram. Для этого внутри telegram найдите @BotFather и следуйте инструкциям. В итоге будет дан токен вашего бота, сохраните его. 
Откройте браузерную панель управления Ouroboros на вкладке Settings (Настройки) и внесите:
- TELEGRAM_BOT_TOKEN — токен вашего Telegram-бота, полученный у @BotFather
