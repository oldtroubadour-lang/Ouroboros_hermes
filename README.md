# Ouroboros_hermes
Этот проект представляет собой Систему поддержки принятия решений (СППР) банка на базе автономного агента Ouroboros. Система в реальном времени мониторит чрезвычайные ситуации, сопоставляет их с геосеткой банковских отделений/банкоматов, формирует тактические сценарии реагирования и высылает интерактивный отчет руководству в Telegram.

(1) Структура файлов и их размещение
-
Для успешного запуска перенесите файлы из репозитория на новую машину в следующие папки:

А. Модуль логики СППР (кастомные навыки)
  Все папки навыков из этого репозитория скопируйте в каталог пользовательских расширений вашего Уробороса
Путь: ~/Ouroboros/data/skills/external/
- temp_storage — общая память и Episodic DB системы.
- telegram-channels - модуль сканирования и, главное, виджет подключения публичных телеграм каналов. 
- incident_ner — сборщик новостей МЧС и классификатор инцидентов.
- asset_geo_matcher — пространственный сопоставитель локаций ЧС и объектов банка.
- risk_scenario_generator — планировщик тактических сценариев.
- telegram-bridge — интерактивный бот управления Hermes.
- dss_orchestrator - оркестратор и организатор совместной работы остальных навыков

Б. Справочники (Активы и Сценарии)
Файлы Excel, находящиеся в корне репозитория GitHub, скопируйте в стандартную папку загрузок текущего системного пользователя:
Путь: C:\Users\<Имя_Пользователя>\Downloads\ (на Windows) или ~/Downloads/ (на macOS/Linux).
- Справочник объектов для агента.xlsx — географический реестр 20 758 банкоматов и офисов (фиктивные данные, все совпадения случайные)
- Агент - примеры сценариев.xlsx — примеры реагирования и решений (в будущем будет RAG)


(2) Быстрый старт (5 простых шагов)
-
Шаг 0. Установка платформы. 
Установите и запустите Ouroboros на новой машине, если еще не установлен и не настроен. 

Шаг 1. Развертывание зависимостей
Для обработки Excel-справочников в окружение Ouroboros должны быть установлены внешние библиотеки. Откройте терминал на новой машине и выполните:
pip install openpyxl edge-tts

Шаг 2. Ввод Telegram-секретов
Создайте бота в telegram. Для этого внутри telegram найдите @BotFather и следуйте инструкциям. В итоге будет дан токен вашего бота, сохраните его. 
Откройте браузерную панель управления Ouroboros на вкладке Settings (Настройки) и внесите:
- TELEGRAM_BOT_TOKEN — токен вашего Telegram-бота, полученный у @BotFather

Шаг 3. Локальный аудит и Активация
Чтобы Ouroboros доверял коду перенесенных модулей, они должны пройти локальный аудит безопасности:
1.  Откройте браузерную панель Ouroboros на панели Skills (Навыки).
2.  Нажмите кнопку Review напротив каждого из наших навыков (система запустит внутренний трехмодельный анализ кода).
3.  После успешного прохождения проверки переключите тумблер в состояние Enable и предоставьте запрашиваемые разрешения во всплывающем окне.

Шаг 4. Привязка публичных телеграм каналов
Для навыка telegram-channels нужно использовать его виджет. В виджете найти поля для добавления публичных телеграм каналов, вставить туда их наименования, нажать применить и убедиться, что они появились в списке подключенных каналов. Альтернативно, можно попросить Уробороса добавить набор каналов на чтение для этого навыка.
Для прототипа даны справочники фиктивных объектов только по территории юга России, поэтому чтобы прототип реально мог находить пересечения - каналы нужно тоже подключать из южных регионов. 
Например: agiprd, rk_gov, chp_checchnya, mchs_95, mchsdagestan, crimea_mchs

Шаг 5. Привязка бота и Тестирование полного цикла
1.  Откройте Telegram, найдите своего бота и отправьте ему команду /start. Бот автоматически привяжет ваш аккаунт (TOFU-авторизация) и в ответ пришлет интерактивную инлайн-панель управления.
2.  Нажмите кнопку 🧪 Тест (или Test Cycle при английской локали) на клавиатуре бота.
3.  Готово! Бот мгновенно запустит внутренний процесс: соберет свежие новости МЧС ➔ классифицирует ЧС ➔ рассчитает по Excel-справочнику число затронутых банкоматов ➔ сформирует планы реагирования и пришлет прямо в чат красиво отформатированный русский отчет с активными кнопками быстрой выгрузки на почту (не подключено в рамках прототипа) или отслеживания (не настроено в рамках прототипа)!

Шаг 6. Настройка и подготовка
Несмотря на то, что формально 5 шагов выше должно быть достаточно для запуска и работы, практика работы с Уроборосом показывает, что правильно запустить авто-ремонт, авто-настройку и подготовку. Именно для этого делался навык оркестратора. 
После включения всех навыков, запуска и присоединения своего телеграм-бота, подключения внешних телеграм каналов для чтения надо сказать Уроборосу, что: 
_У тебя есть набор навыков, которые комбинируются в систему поддержки принятия решений. Проверь их совместную работоспособность, произведи между ними настройки, убедись что система работает_.
